# Click to Remove Circles

A simple mini project: three blue circles are displayed in a row.  
When you click a circle, it is removed from the DOM completely (not just hidden).

## Features
- 3 blue circles displayed in a single row
- Click a circle to remove it from the document (DOM removal)
- Uses the parent's `children` collection to validate and remove clicked elements

## Tech Stack
- HTML
- CSS
- JavaScript (DOM)

## How to Run
1. Download / clone the project
2. Open `index.html` in your browser
